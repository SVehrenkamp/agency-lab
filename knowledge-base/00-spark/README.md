# 00 · Spark

This folder holds **one** file: `spark.md`, the project's origin record — the promoted
idea, frozen at the moment it graduated into its own repo. It's the single seed the
Kickoff phase reads.

- **Consumes:** the promoted idea (carried in at graduation)
- **Produces:** `spark.md` (one paragraph + the problem it chases)
- **Default mode:** `mode:collab`

## Spark is a lab-level concept, not a project phase

A repo is one idea, so there's no backlog to triage *here*. The cheap "many sparks, most
never go further" filter happens at the **lab** level — as `spark`-labeled issues on your
Agency Lab home repo. **Graduation** (promoting a spark) is the same act as creating this
project from the template; by the time this repo exists, the spark has already won. That's
why there's no `00 · Spark` milestone in a project — the pipeline starts at Kickoff.

Start `spark.md` from `../_templates/_artifact-template.md`.
