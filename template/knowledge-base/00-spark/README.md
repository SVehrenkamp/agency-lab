# 00 · Spark

The cheap front door. An idea gets one paragraph and a gut-check — nothing more.
Most sparks die here, and that is the point: this stage protects experimentation
speed by keeping the heavyweight pipeline away from ideas that haven't earned it.

- **Consumes:** a raw idea
- **Produces:** `spark.md` (one paragraph + gut-check)
- **Default mode:** `mode:collab`
- **Exit:** the **graduation gate** — only a consciously promoted spark advances to Kickoff.

Start from `../_templates/_artifact-template.md`.
