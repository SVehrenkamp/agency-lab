# 09 · Review / QA

Close out each feature delivery with thorough testing and review against
requirements before it is considered done. Checklists live in `review-checklists/`.

- **Consumes:** each delivered feature
- **Produces:** `qa-reports.md`, `review-checklists/*`
- **Default mode:** `mode:collab` (sign-off)
- **Backing skill:** `engineering:testing-strategy`
